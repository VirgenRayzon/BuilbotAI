# **App Name**: Forge Architect AI

## Core Features:

- AI Build Advisor Chatbot: Engage with an intelligent AI tool that recommends compatible PC components based on your specific needs, budget, and desired performance, providing personalized build strategies.
- Interactive Component Selector: Browse and select various PC components (CPU, GPU, Motherboard, etc.) from an up-to-date catalog.
- Real-time Compatibility Checker: Automatically verifies the compatibility of selected parts as you add them to your build, highlighting any potential conflicts.
- Estimated Build Cost Display: View a running total of the estimated cost of your PC build, updated instantly with each component selection.
- Build Summary and Sharing: Generate a comprehensive summary of your proposed PC build, with options to save or share your configuration.

## Style Guidelines:

- Primary color: A sophisticated, deep blue (#448FC4) to convey technology and trust.
- Background color: A very dark, desaturated blue (#21262B) providing a clean, high-contrast canvas appropriate for a sleek tech aesthetic.
- Accent color: A light, cool silver-grey (#CDD8DB) for highlights, interactive elements, and textual contrast, analogous to the primary blue while offering excellent readability.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, tech-inspired feel that resonates with a PC building theme.
- Body font: 'Inter' (sans-serif) for its high readability and neutral, objective appearance, ideal for displaying specifications and instructional text.
- Utilize minimalist, modern line icons for all interactive elements and component categories, enhancing the sleek and technical aesthetic.
- Employ a clean, structured, and responsive layout with ample whitespace to ensure focus on components and recommendations, optimizing for both desktop and mobile displays.
- Implement subtle, fluid animations for transitions between selections, component additions, and chatbot interactions to provide a premium and responsive user experience.
